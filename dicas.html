Dicas Sobre o git

git clone https://link-com-o-nome-do-repositório

Esse comando vai baixar o projeto para a sua máquina

git status

Verificar o que foi alterado para enviar ao repositório, geralmente usado antes do git commit.

git add .

Usado para adicionar tudo o que foi alterado no git commit (comando a seguir).

git commit -a -m "mensagem do commit"

Após alterações, esse comando irá enviar um "commit", ou seja salvar tudo o que foi alterado e preparar esse conteúdo para enviar ao repositório

git push

Esse comando é obrigatório passar usuário e senha, após adicionar usuário e senha, tudo o que foi commitado será enviado ao repositório

git pull

Caso queira garantir que os arquivos da sua máquina estão atualizados, usar esse comando para baixar tudo

git branch nome-da-branch

Esse comando criará uma branch em seu local de trabalho.

git checkout nome-da-branch

Comando para trocar de branch.

git merge

Comando usado para mesclar 2 branches.
